# AI News Open v1.0.0 Announcement

## Primary Chinese Announcement

AI News Open `v1.0.0` 正式发布。

这是一个面向真实工作流的开源项目，用来把分散的国内外 AI 新闻整理成一条可发布的中文日报链路：采集、清洗、去重、正文提取、国际新闻中文翻译、日报生成、存档和多渠道发布。

这次进入 `v1.0.0`，不只是功能叠加，而是把一批真正影响长期可用性的工程边界钉住了：

- 配置项、CLI、HTTP API、导出 JSON 和数据库升级路径有了明确的 `v1.x` 兼容承诺
- 默认源和正文抽取行为补上了回归测试
- 发布链路默认具备幂等保护，同一个已存档日报不会被重复发布到同一个目标
- 导出结果和数据库状态都带上了版本语义，升级和排障更可控
- 部署、迁移、排障文档已经补齐，仓库不再只是“代码能跑”，而是“别人能接手”

项目当前支持：

- 国内外 RSS / Atom AI 新闻采集
- 文章清洗、去重、正文抽取
- OpenAI-compatible LLM 的中文翻译与摘要生成
- 中文 AI 日报生成与历史存档
- Telegram、飞书、静态站点、微信公众号草稿 / 发布工作流
- FastAPI API、CLI、零构建管理后台

## GitHub / LinkedIn Style English Post

AI News Open `v1.0.0` is out.

It is an open-source workflow for turning domestic and global AI news into a publishable Chinese daily digest, with ingestion, cleanup, extraction, translation, digest generation, archiving, and multi-channel publishing in one stack.

`v1.0.0` is the first release that locks the project’s `v1.x` contract across config, CLI, HTTP API, export payloads, and SQLite upgrade behavior.
